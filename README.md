# TeaTour-ssm
使用Spring+SpringMVC+MyBatis+Java+JSP+Tomcat+MySQL完成的商城网站，具有茶类商品购买、评价和茶园体验预约等功能<br>
配置须知：<br>
数据库：MySQL5.1.38<br>
服务器：Tomcat9.0<br>
jdk：javase1.8<br>
以下根据自己的需要修改<br>
访问端口：8888<br>
数据库端口：3306（一般都是3306）<br>
数据库用户：root<br>
数据库密码：123456<br>
项目本地访问地址：http://localhost:8888/TeaTour/home<br>
